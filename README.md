# Prettier config
> Prettier config for multiple project with the same format

## Getting started

### Installation
```bash
npm install --save-dev @kontentino/prettier-config
```

### Usage
*prettier.config.js*
```javascript
module.exports = require('@kontentino/prettier-config/prettier.config');
```
